# Tic-Tac-Toe 5x5 AI

## Overview

A neutral 5x5 tictactoe ai using enhanced minimax algorithm with alpha-beta pruning, score evaluation, adaptive max depth

## Setup

```bash
git clone https://github.com/ducvu0907/tictactoe-ai
cd tictactoe-ai
python game.py
```
